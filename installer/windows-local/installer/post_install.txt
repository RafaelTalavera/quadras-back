Checklist de post-instalacion:
1. El instalador debe ejecutar scripts/install_local_stack.ps1 al finalizar.
2. Si se reintenta manualmente, usar scripts/install_local_stack.ps1.
3. Verificar config/application-local.properties.
4. Validar GET http://127.0.0.1:8080/api/v1/system/health.
5. Abrir costanorte.exe y hacer login con operador.demo.
6. Para apagar la instalacion local, usar scripts/stop_local_stack.ps1.
7. Logs y config generada quedan en C:\ProgramData\CostanorteLocal.
